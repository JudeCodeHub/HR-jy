# MASTER HR - Complete Setup and Run Guide

Follow these steps in order to set up and run the application on your laptop.

---

## Phase 1: Prerequisites & Installations
Make sure the following are installed on your laptop before starting:
1. **Node.js** (version 18 or higher): Download from [nodejs.org](https://nodejs.org/)
2. **Python 3.10**: Download from [python.org](https://www.python.org/) 
   *(IMPORTANT: Check the box "Add Python to PATH" during installation).*
3. **Expo Go App**: Download "Expo Go" on your physical iOS or Android phone from the App Store / Play Store.

---

## Phase 2: Local IP Address Setup (Crucial)
Since you will be running the app on a physical phone, the phone needs to connect to the servers running on your laptop.

1. Open **Command Prompt** (cmd) on your laptop.
2. Run the command:
   ```cmd
   ipconfig
   ```
3. Locate your active Wi-Fi adapter and copy the **IPv4 Address** (it will look like `192.168.x.x` or `172.x.x.x`).
4. In the `Frontend` directory, copy/rename `.env.example` to `.env` and update it with your laptop's IP address:
   ```env
   EXPO_PUBLIC_API_URL=http://<YOUR_IP_ADDRESS>:3501
   EXPO_PUBLIC_API_URL2=http://<YOUR_IP_ADDRESS>:5000
   ```
   *(For example: `EXPO_PUBLIC_API_URL=http://192.168.1.15:3501`)*

---

## Phase 3: Start Node.js Backend Server
1. In the `Backend` directory, copy/rename `.env.example` to `.env` (it contains the database connection string).
2. Open a terminal/command prompt and navigate to the `Backend` directory:
   ```cmd
   cd Backend
   ```
3. Install the backend dependencies:
   ```cmd
   npm install
   ```
4. Start the backend server:
   ```cmd
   npm start
   ```
   *(Keep this terminal open. The backend runs on port 3501).*

---

## Phase 4: Start Face Recognition Server (AI)
1. Open a **new** terminal window and navigate to the face recognition directory:
   ```cmd
   cd Backend/FaceRecognation
   ```
2. Create a clean Python virtual environment (do not use the previous developer's venv folder):
   ```cmd
   python -m venv venv
   ```
3. Activate the virtual environment:
   * **Windows (Command Prompt):** `venv\Scripts\activate`
   * **Windows (PowerShell):** `.\venv\Scripts\Activate.ps1`
   * **Mac/Linux:** `source venv/bin/activate`
4. Install the required Python packages:
   ```cmd
   pip install flask flask-cors numpy face-recognition tensorflow pillow opencv-python
   ```
5. Start the Face Recognition server:
   ```cmd
   python App.py
   ```
   *(Keep this terminal open. The AI service runs on port 5000).*

---

## Phase 5: Start Frontend App (Expo Mobile)
1. Open a **new** terminal window and navigate to the `Frontend` directory:
   ```cmd
   cd Frontend
   ```
2. Install the mobile frontend dependencies:
   ```cmd
   npm install
   ```
3. Start the Expo builder:
   ```cmd
   npx expo start --clear
   ```
4. Connect your phone and laptop to the **same Wi-Fi network**.
5. Open the **Expo Go** app on your phone and scan the QR code displayed in the terminal.

---

## Phase 6: How to Test the Complete Flow
Once all three servers (Backend, AI, and Expo) are running:
1. **Sign Up as HR**: Open the app on your phone, choose **HR**, and click **Sign Up** to create an HR account.
2. **Set Office Location**: Log in as the HR, and fetch/set the Office Location. This establishes the office boundary geofence.
3. **Register an Employee**: While logged in as HR, go to **Manage Employees** -> **Add Employee**. Upload a clear photo of the employee's face.
4. **Compile Face Encodings (AI Database)**:
   * Go back to the Face Recognition terminal (`Backend/FaceRecognation`).
   * Press `Ctrl + C` to stop the Python server.
   * Run the compiler script:
     ```cmd
     python encodings.py
     ```
     *(This reads the new employee's photo from the database and saves their facial features).*
   * Restart the AI server:
     ```cmd
     python App.py
     ```
5. **Mark Attendance**:
   * Log out of the HR account in the mobile app.
   * Log in as the newly created **Employee**.
   * Tap **Mark Attendance**.
   * The app will check if you are within the office boundary, enable the camera, take your selfie, and mark your attendance successfully!
